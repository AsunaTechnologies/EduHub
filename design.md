# Assyl DataHub - Design Document

## Design Philosophy

### Visual Language
- **Futuristic Sci-Fi Aesthetic**: Clean, minimal interface with advanced technological elements
- **Data-Driven Design**: Information architecture that emphasizes clarity and accessibility
- **Professional Authority**: Trustworthy design that conveys expertise in educational consulting
- **Cultural Sensitivity**: Design that works across Kazakh, Russian, and English contexts

### Color Palette
- **Primary**: Deep Space Blue (#0B1426) - main background
- **Secondary**: Electric Cyan (#00D4FF) - accents and highlights
- **Tertiary**: Warm White (#F8FAFC) - text and content areas
- **Accent**: Neon Green (#39FF14) - interactive elements and CTAs
- **Neutral**: Silver Gray (#94A3B8) - secondary text and borders

### Typography
- **Display Font**: "Orbitron" - futuristic, geometric sans-serif for headings
- **Body Font**: "Inter" - clean, highly readable sans-serif for content
- **Monospace**: "JetBrains Mono" - for data displays and technical information

### Layout Principles
- **Grid System**: 12-column responsive grid with consistent spacing
- **Minimal UI**: Clean interfaces with purposeful white space
- **Hierarchical Information**: Clear visual hierarchy for complex data
- **Mobile-First**: Responsive design starting from mobile breakpoints

## Visual Effects

### Used Libraries
1. **Anime.js**: Smooth micro-interactions and page transitions
2. **Three.js**: 3D virtual campus tours and data visualizations
3. **ECharts.js**: University comparison charts and analytics
4. **Splitting.js**: Advanced text animations for headings
5. **Pixi.js**: Particle effects and background animations
6. **Typed.js**: Typewriter effects for hero text
7. **Splide**: Image carousels and university galleries

### Animation Strategy
- **Subtle Motion**: Gentle, purposeful animations that enhance UX
- **Loading States**: Engaging loading animations for data-heavy sections
- **Micro-interactions**: Hover effects and button feedback
- **Page Transitions**: Smooth navigation between sections

### Header Effects
- **Animated Background**: Subtle particle system using Pixi.js
- **Gradient Overlays**: Dynamic color transitions
- **Parallax Elements**: Layered movement for depth
- **Data Visualization**: Real-time statistics display

### Interactive Elements
- **3D Campus Tours**: Immersive virtual exploration
- **Comparison Tools**: Dynamic side-by-side university analysis
- **Profile Dashboards**: Personalized user interfaces
- **Language Switcher**: Smooth transitions between locales

### Styling Approach
- **CSS Custom Properties**: Easy configuration for non-technical users
- **Component-Based**: Modular design system
- **Accessibility First**: WCAG compliant color contrast and navigation
- **Performance Optimized**: Efficient loading and rendering

### Background Treatment
- **Consistent Theme**: Unified dark theme throughout
- **Subtle Patterns**: Geometric elements and grid overlays
- **Depth Layers**: Multiple z-index levels for visual hierarchy
- **Responsive Scaling**: Adaptive sizing for all screen types
